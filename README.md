# Supported DLC Types
- [ ] Special Transport
- [x] All Map DLCs*
- [x] Cargo DLCs*
- [x] Truck Tuning DLCs (HS DLC, Paints, Flags, etc.)
- [x] Trailer DLCs (Krone, Schmitz, etc.)

*Map DLCs require you to obtain the .scs files yourself.<br>
*All Cargo DLCs work, exluding Special Transport.
